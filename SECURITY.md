# Security Infrastructure

Moonshine Capital infrastructure prioritizes data sovereignty and privacy.

## Telemetry Security

- All email data is masked (hashed) before reaching our telemetry ledgers.
- Serverless ingest points are rate-limited via Vercel Edge protection.
- No PII is stored in plain-text logs.

## Reporting Vulnerabilities

If you identify a security gap in the infrastructure, contact: security@distilledfunding.com
